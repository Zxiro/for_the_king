## Quick Start
### 安裝vcpkg
1. git clone https://github.com/microsoft/vcpkg.git
2. cd vcpkg
3. bootstrap-vcpkg.bat
4. .\vcpkg.exe integrate install
5. set VCPKG_ROOT="C:\path\to\vcpkg"
6. set PATH=%VCPKG_ROOT%;%PATH%
7. vcpkg install ftxui

Visual Studio 編譯並執行
